# OutlookPanel

基于 PHP + OAuth2 IMAP 的现代化 Outlook 邮件管理面板

---

## 📖 项目介绍

OutlookPanel 是一个轻量化的 Outlook / Hotmail 邮件管理系统。

项目基于：

- PHP
- SQLite
- OAuth2 IMAP

开发，支持多账户管理、邮件读取、HTML 邮件渲染、JSON 导入导出、Telegram Bot 备份等功能。

适用于：

- Outlook 邮箱聚合管理
- 多账户邮件查看
- 轻量化 WebMail
- 邮件监控
- 开发测试环境
